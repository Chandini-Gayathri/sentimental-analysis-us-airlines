
# Sentiment Analysis on US Airline Tweets

## Overview
This project focuses on performing sentiment analysis on tweets related to US airlines.
The goal is to classify customer sentiment as positive, negative, or neutral using
Natural Language Processing (NLP) and machine learning techniques.

## Dataset
- Source: US Airline Twitter Dataset
- Contains tweets labeled with sentiment categories
- Preprocessing includes text cleaning, tokenization, and vectorization

## Approach
- Data preprocessing and text cleaning
- Feature extraction using TF-IDF
- Model training and evaluation
- Visualization of sentiment distribution

## Technologies Used
- Python
- Pandas, NumPy
- NLTK / Scikit-learn
- Matplotlib / Seaborn
